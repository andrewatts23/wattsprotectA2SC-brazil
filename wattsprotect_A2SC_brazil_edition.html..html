⚖️ Manifesto

WattsProtect™ A²SC é a cadeia viva de custódia que transforma cada decisão de IA em evidência. Segurança protege. Prova lembra. Soberania exige as duas.

Desenvolvido para ambientes soberanos e críticos, o A²SC sela o comportamento dos sistemas em atestados criptográficos verificáveis — sem telemetria, sem vazamento de dados, apenas verdade matemática. Cada evento se torna uma prova auditável, cada política um contrato vivo.

🔹 Estrutura de Custódia A²SC

1) Evento
Captura do gatilho ou decisão do sistema em tempo real.
2) Política
Regras OPA/Rego aplicadas antes da execução.
3) Atentação
TPM/TEE valida integridade do ambiente.
4) Encadeamento
Hashes Merkle vinculam cada registro ao anterior.
5) Pacote de Prova
Evento + política + assinatura selados.
6) Cofre Imutável
Armazenamento forense auditável e soberano.

💻 Código Exemplar

policy.yaml

# CAL-APPROVAL-001
versão: 1
política_id: CAL-APPROVAL-001
descrição: "Mudanças de calibração exigem treinamento, SoD e deriva dentro da tolerância."
controles:
  - id: TREINAMENTO_VALIDO
    requer: ator.treinamento.calibracao == true
  - id: SOD_ENFORCADA
    requer: ator.id != solicitacao.solicitado_por
  - id: LIMITE_DERIVA
    requer: metricas.deriva_ppm <= parametros.max_ppm
parametros:
  max_ppm: 25
aprovações:
  - papel: "Supervisor"
    mfa: true

gate.rego

pacote portoes.calibracao
padrão permitir = falso
negar[msg] {
  deriva_futura := entrada.simulacao.deriva_ppm
  deriva_futura > entrada.parametros.max_ppm
  msg := sprintf("deriva prevista acima do limite: %v > %v", [deriva_futura, entrada.parametros.max_ppm])
}
permitir { count(negacao) == 0 }
